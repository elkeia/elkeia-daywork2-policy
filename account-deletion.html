정책 안내

계정 및 데이터 삭제 안내

최종 업데이트: 2026-06-25 · 문의: [문의 이메일]

당직월력은 앱 안과 앱 밖에서 모두 계정 삭제 요청을 시작할 수 있도록 안내합니다. Google Play Console의 계정 삭제 URL에는 이 페이지 주소를 입력하세요.

1. 앱 안에서 삭제하기

  1. 이메일 계정으로 로그인합니다.
  2. 홈 화면의 설정 탭에서 세션 영역으로 이동합니다.
  3. 회원 탈퇴를 선택합니다.
  4. 확인 문구와 비밀번호를 입력해 탈퇴를 완료합니다.

그룹장은 먼저 그룹장 위임을 완료해야 탈퇴할 수 있습니다. 이는 그룹 운영 데이터가 갑자기 관리 불능 상태가 되는 것을 막기 위한 제한입니다.

2. 웹에서 삭제 요청하기

앱을 삭제했거나 앱에 접속할 수 없는 경우 [문의 이메일]로 삭제 요청을 보낼 수 있습니다.

  • 메일 제목: 당직월력 계정 삭제 요청
  • 본문 필수 정보: 가입 이메일, 요청자 이름, 소속 그룹을 식별할 수 있는 정보
  • 본인 확인 후 계정 삭제 또는 추가 확인 절차를 안내합니다.

웹 요청 주소: [문의 이메일]

3. 삭제되는 데이터

  • Supabase Auth 이메일 계정
  • 개인 나무 성장 데이터와 보관함 보상 기록
  • 광고 보상 nonce 등 중복 지급 방지용 개인 기록
  • 이메일 계정과 그룹원 멤버십의 인증 연결
  • 해당 이메일이 그룹 대표 이메일로 남아 있는 참조 정보
  • 기기 푸시 토큰 중 계정과 연결된 항목

4. 유지될 수 있는 데이터

당직표는 팀 공동 운영 기록이므로 계정 삭제 후에도 그룹 안에 남을 수 있습니다. 예를 들어 월별 근무표, 변경 요청, 제외 요청, 승인/반려 상태, 건의 내용, 그룹 설정은 다른 그룹원이 계속 확인해야 하는 운영 데이터일 수 있습니다. 필요한 경우 그룹장에게 그룹 데이터 수정 또는 삭제를 별도로 요청하세요.

5. 처리 기간

앱 내 회원 탈퇴는 서버 처리 완료 후 즉시 반영됩니다. 이메일을 통한 수동 요청은 본인 확인과 필요한 검토 후 지체 없이 처리하며, 추가 확인이 필요한 경우 요청자에게 진행 상황을 안내합니다.
